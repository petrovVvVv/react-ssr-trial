# react-ssr-trial
React server-side rendering sample project.

## Running the SSR

- `npm install`
- `npm run build` - build the front-end (currently need to rebuild each time when changes are done)
- `npm run start:ssr`
- Open localhost:9000

Has:
- loading of static pages, e.g. the ones in `content_1.json`
- Home Page
- Redirection to 404
